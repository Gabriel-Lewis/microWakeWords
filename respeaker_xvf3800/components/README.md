# ReSpeaker XVF3800 ESPHome Components

Vendored ReSpeaker XVF3800 hardware components used by `respeakerXVF3800.yaml`.

Upstream source: formatBCE/Respeaker-XVF3800-ESPHome-integration.

These files are kept in-tree so the ReSpeaker XVF3800 firmware does not depend on a separate XVF3800 ESPHome component repo at build time.
